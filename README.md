# Sistema de Controle de Estoque para Ferramentas

Sistema web desenvolvido para o gerenciamento de estoque de ferramentas, permitindo o cadastro de produtos, controle de entradas e saídas, autenticação de usuários, geração de relatórios e visualização de indicadores por meio de um dashboard.

## Sobre o Projeto

Este projeto foi desenvolvido como **atividade avaliativa do curso de Análise e Desenvolvimento de Sistemas**, tendo como objetivo aplicar na prática os conceitos estudados durante a graduação, incluindo Engenharia de Software, Desenvolvimento Web, Banco de Dados e Arquitetura em Camadas.

O sistema foi projetado para atender aos requisitos funcionais e não funcionais propostos, oferecendo uma solução para controle de estoque com registro de movimentações, autenticação de usuários e geração de informações gerenciais.
